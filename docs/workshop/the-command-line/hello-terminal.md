# Hello Terminal

1. Open your terminal on your computer and type the following command:
```bash
echo "Hello Terminal"
```
2. Press ++enter++

???+ question "What happens when you run `echo "Hello Terminal"`?"
	1. `Hello Terminal` printed to the terminal's console
	2. An error message printed to the terminal's console
	3. The terminal printed nothing and is waiting for a new input

	??? success "Solution"
        **1** &rarr; `Hello Terminal` printed to the terminal's console 

		---
		
   		**Explanation:** `echo` prints its arguments to the terminal.
